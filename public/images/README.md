# Images Folder Structure

## How to Use

1. **Hero Images** (`public/images/hero/`)
   - Add couple photos for the hero section
   - Recommended: 2-3 high-quality images
   - Formats: JPG, PNG, WebP

2. **Gallery Images** (`public/images/gallery/`)
   - Add all your personal photos here
   - These will be displayed in the photo gallery
   - No limit on number of images
   - Formats: JPG, PNG, WebP

3. **Couple Photos** (`public/images/couple/`)
   - Special photos of the couple
   - Used in specific sections
   - Formats: JPG, PNG, WebP

## Image Naming
- Use descriptive names: `photo-1.jpg`, `couple-smile.jpg`, etc.
- Avoid spaces in filenames (use hyphens or underscores)

## Image Optimization Tips
- Recommended size: 1920x1080px or smaller
- Compress images before adding (use tools like TinyPNG)
- Keep file sizes under 500KB for faster loading
